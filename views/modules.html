Creating modules

Modules can be a set of services, or an entire application. Just provide a module name followed by an array of dependencies to the angular.module function. Sometimes both are required.

Copy 
angular.module('hydra',['hydra.services','hydra.directives']);
angular.module('hydra.services',[]);
angular.module('hydra.directives',[]);

Creating services

Services are just syntax sugar for angular providers. But provider syntax is usually overkill. This service will provide data from some api.

Copy 
hydra.service('TripsService',['$http',function($http){
	var TripsService = {};
	TripsService.getTrips = function (callback) {
		$http.get('/trips').success(function(trips){
			callback(trips);
		});
	}
	return TripsService;
}]);

When the TripsService is called with getTrips, it will callback a function and pass through trips.
