Getting started

An overview of Jeffy, how to download and use, basic templates and examples, and more.

Jeffy (currently v0.2.0) has a few easy ways to quickly get started, each one appealing to a different skill level and use case. Read through to see what suits your particular needs.

Cygwin

Run setup-x86_64.exe any time you want to update or install a Cygwin package for 64-bit windows. The signature for setup-x86_64.exe can be used to verify the validity of this binary using this public key.

Download Cygwin

Git SCM

Git for Windows focuses on offering a lightweight, native set of tools that bring the full feature set of the Git SCM to Windows while providing appropriate user interfaces for experienced Git users and novices alike.

Download Git SCM

Node.js

Node.js is a platform built on Chrome's JavaScript runtime for easily building fast, scalable network applications. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient.

Download Node.js

Jeffy Package

The folks over at GCC graciously provide a package for Jeffy's dependencies. Just use this Jeffy Package link.

$ curl https://github.com/braungoodson/jeffy/package

Install with git

Clone and install Jeffy's reopository using git.

$ git clone https://github.com/braungoodson/jeffy

That's it!

Jeffy is easy to use, easy to follow, and provides a platform for rapid front-end solutions.
